## 买了之后怎么激活

很简单，四步搞定：

1. 点击「开始菜单」，搜索并打开「设置」
2. 进入「系统 → 激活」
3. 点击「激活 / 更改」按钮
4. 输入购买后收到的激活码，点击「下一步」等待激活完成

整个过程五分钟以内。激活完成后建议登录微软账号，这样数字权利就绑在账号上了，下次换机直接登录账号就能自动激活，不需要再输激活码。
